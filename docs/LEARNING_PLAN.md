# LearnCpp C++ 学习指南

这是一个基于 LearnCpp.com.cn 的完整 C++ 学习路径。

## 学习路线图概览

### 阶段一：入门基础（第0-2章）
- **第0章**：简介/入门 - 安装开发环境，编写第一个程序
- **第1章**：C++基础 - 语句、注释、变量、输入输出
- **第2章**：函数和文件 - 函数定义、多文件编程、头文件

### 阶段二：核心概念（第3-8章）
- **第3章**：调试 - 使用调试器查找和修复错误
- **第4章**：基本数据类型 - 整型、浮点型、布尔、字符
- **第5章**：常量和字符串 - const、constexpr、std::string
- **第6章**：运算符 - 算术、逻辑、位运算
- **第7章**：作用域和生命周期 - 变量作用域、全局变量
- **第8章**：控制流 - if/switch、循环、随机数

### 阶段三：深化理解（第9-15章）
- **第9章**：错误处理 - 断言、异常处理
- **第10章**：类型系统 - 类型转换、typedef、auto
- **第11章**：函数模板 - 模板函数、重载
- **第12章**：引用和指针 - 左值引用、指针、std::optional
- **第13章**：枚举和结构体 - 自定义类型
- **第14章**：类（一）- 构造函数、访问控制
- **第15章**：类（二）- 析构函数、静态成员、友元

### 阶段四：标准库（第16-20章）
- **第16章**：std::vector - 动态数组
- **第17章**：std::array 和 C风格数组
- **第18章**：迭代器和算法
- **第19章**：动态内存分配
- **第20章**：函数指针和Lambda

### 阶段五：面向对象高级（第21-25章）
- **第21章**：运算符重载
- **第22章**：移动语义和智能指针
- **第23章**：对象关系 - 组合、聚合、关联
- **第24章**：继承
- **第25章**：虚函数和多态

### 阶段六：模板和进阶（第26-28章）
- **第26章**：模板类
- **第27章**：异常处理
- **第28章**：输入输出流

### 附录
- **附录A**：库的使用
- **附录B**：C++11/14/17/20/23 新特性

## 学习建议

1. **动手实践**：每个示例都亲手输入并运行
2. **完成练习**：每章末尾的测验和练习题一定要做
3. **循序渐进**：不要跳过基础直接学高级内容
4. **温故知新**：定期回顾之前的内容
5. **项目驱动**：学完后尝试编写小项目巩固知识

## 练习项目建议

完成基础学习后，可以尝试以下项目：

1. **计算器程序** - 巩固函数和运算符
2. **学生成绩管理系统** - 巩固类和结构体
3. **文本编辑器** - 巩固字符串和文件I/O
4. **简单的游戏** - 巩固面向对象编程
